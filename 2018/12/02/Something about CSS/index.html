Something About CSS

Posted on 2018-12-02

这个文章记录了我在学习CSS的过程中得到的一些体会和一些注意事项。

注意事项

关于命名

类和ID的命名最好不要以数字开头。

以数字开头的 class 和 ID 在 Mozilla/Firefox 浏览器中不起作用。

关于多重样式的优先级

一般来说优先级如下：

内联样式 > 内部样式 > 外部样式 > 浏览器默认样式

但是，当我们在 \<head> 部分定义外部样式和内部样式时，若将内部样式写在了前面，则多重样式选择时，外部样式会覆盖掉内部样式。
